## 下载

### Docker

```
docker pull apachecn0/freelearn-quant-zh
docker run -tid -p <port>:80 apachecn0/freelearn-quant-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-quant-zh
freelearn-quant-zh <port>
# 访问 http://localhost:{port} 查看文档
```
